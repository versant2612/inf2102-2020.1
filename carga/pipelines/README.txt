Passos para download do pipeline

1) Conectar com o servidor do Linked Pipes ETL 

ssh -L 8080:127.0.0.1:8080 cloud-di@xxxxxxxx.cloud.inf.puc-rio.br

2) Entrar em http://localhost:8080/#/pipelines

3) Clicar nos ... em cada pipeline e selecionar Download

Importante manter uma cópia das configurações do pipeline aqui no Git para o caso de ser necessário rastrear as alterções um processo de carga ou voltar uma vesão anterior já que a ferramenta Linked Pipes ETL não faz controle de versão
